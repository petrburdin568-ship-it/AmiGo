# AmiGo Song Comments (GitHub Pages MVP)

Статический прототип для GitHub Pages: текущая песня берётся из фиктивной функции, а комментарии хранятся в браузере (localStorage). Никаких внешних API и серверной части.

## Как включить GitHub Pages

1) Залей репозиторий на GitHub.
2) В репозитории: **Settings → Pages**.
3) **Build and deployment** → **Deploy from a branch**.
4) Выбери ветку `main` и папку `/ (root)`.

После деплоя сайт будет доступен по адресу вида:
`https://<username>.github.io/<repo>/`

## Локальный просмотр

Можно просто открыть `index.html` двойным кликом.
Если хочешь через локальный сервер (чтобы было “как на хостинге”) — используй любой статический сервер.

## Ограничения GitHub Pages

GitHub Pages не запускает Python/FastAPI/SQLite. Поэтому в этом MVP:
- “Текущая песня” — имитация (меняется раз в ~30 секунд).
- Комментарии сохраняются только у тебя в браузере (localStorage), не синхронизируются между пользователями.
