# AmiGo — Чаты по интересам (GitHub Pages Demo)

Демо **без сервера** для GitHub Pages:
- регистрация профиля (ник + интересы)
- кнопка **«Найти собеседника»** подбирает профиль с похожими интересами
- 1:1 диалог (сообщения в `localStorage`)
- “реалтайм” между вкладками/окнами на одном компьютере через `BroadcastChannel`

## Как включить GitHub Pages

1) Залей репозиторий на GitHub.
2) **Settings → Pages**
3) **Deploy from a branch**
4) Branch: `main`, Folder: `/ (root)`

## Локальный просмотр

Можно просто открыть `index.html` двойным кликом.

## Ограничения

- Другие пользователи по ссылке не увидят твои сообщения: нет сервера/БД.
- Для “двух людей” в демо: открой 2 вкладки, зарегистрируй 2 профиля и пиши друг другу.
- Для настоящего мультиплеера нужен отдельный backend (и тогда фронт можно оставить на Pages).
